# What To Do

## About
- React.js, react-router, express, webpack, firebase를 사용하여 만든 ToDoApp
- firebase를 이용하여 구글 로그인 연동, firebase 실시간 데이터베이스 연동

## Installation

```
$ npm install -g webpack babel
$ npm install
```


## run scripts

### build

```
$ npm run build
```

### start with production

```
$ npm run start
```


### start with development

```
$ npm run development
```
